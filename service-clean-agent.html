Clean Agent Extinguishing System

Clean Agent is stored in cylinders as a liquid. The cylinders are pressurized with nitrogen which acts as a propelling mechanism for the discharge of the agent. As the agent reaches the discharge nozzle it is vaporized and floods the hazard area as a gas. A CLEAN AGENT system can provide an effective fire extinguishing medium with modular hardware that requires minimal space for installation and a most effective means of fire suppression
